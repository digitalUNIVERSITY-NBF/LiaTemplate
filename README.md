# LiaTemplate

This repository contains the base template for digitalUNIVERSITY courses created in LiaScript.

It contains styling and macro definitions.

## Usage

To apply the template in your course, use one of the following versions in the LiaScript header of your course. 

**Option 1: Specific Version:** 

Replace [Version] by the version you want to use (e.g. v1.0.1)

```
import:     https://raw.githubusercontent.com/digitalUNIVERSITY-NBF/LiaTemplate/[Version]/template.md
```

**Option 2: Latest Version:** 

Replace [Version] by `main`

```
import:     https://raw.githubusercontent.com/digitalUNIVERSITY-NBF/LiaTemplate/main/template.md
```
